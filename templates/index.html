Welcome to the Forex Prediction API

This API predicts EUR/USD forex prices using machine learning models.

API Endpoints

/predict - POST - Predicts the EUR/USD forex prices based on the input data.

Usage

Send a POST request to the /predict endpoint with the following JSON format:

        [
            {
                "Ticker": "EURUSD",
                "Date": 20240527,
                "Time": 150000,
                "Open": 1.0740,
                "High": 1.0750,
                "Low": 1.0730,
                "Close": 1.0745,
                "Volume": 1000
            },
            ...
        ]

The API will return a JSON response with the predicted prices.
